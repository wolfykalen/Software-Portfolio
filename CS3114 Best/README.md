# CS3114 Best Projects - Advanced Data Structures

**Course**: CS3114 - Advanced Data Structures & Algorithms  
**Focus**: Complex algorithms, spatial data structures, optimization, large-scale data processing  
**Author**: Kalen Dacosta (kalendaco)

## 📋 Overview

These 4 projects demonstrate mastery of advanced data structures and algorithms, including sparse matrices, k-d trees, radix sort, and complex spatial indexing. Each project handles large-scale data efficiently and includes comprehensive testing.

## 🎯 Projects

### 1. F25P1MovieRater - Sparse Matrix with Orthogonal Lists
**Description**: Movie rating system using sparse matrix implementation with orthogonal lists for efficient storage and similarity calculations.

**Key Features**:
- **Sparse matrix** using orthogonal lists (row and column headers)
- Efficient memory usage for sparse data
- Movie and reviewer similarity calculations
- CRUD operations for ratings
- Comprehensive unit tests

**Skills Demonstrated**:
- Advanced data structure design
- Memory-efficient algorithms
- Sparse data handling
- Orthogonal list implementation

**Why This Impresses Recruiters**:
Sparse matrices are used in real-world applications (recommendation systems, scientific computing). This shows ability to design efficient data structures for specific use cases.

---

### 2. F25P2GIS - 2D K-D Tree for Geographic Information
**Description**: Geographic Information System using 2D k-d tree for efficient spatial queries and range searches.

**Key Features**:
- **2D k-d tree** implementation for spatial indexing
- Efficient insertion, deletion, and search operations
- Range search for geographic queries
- BST for name-based indexing
- Comprehensive unit tests

**Skills Demonstrated**:
- Spatial data structures
- Tree algorithms
- Geographic data processing
- Multi-index design (name + coordinates)

**Why This Impresses Recruiters**:
K-d trees are used in databases, GIS systems, and machine learning. This demonstrates understanding of spatial indexing and efficient geometric algorithms.

---

### 3. F25P3Radix - LSD Radix Sort on Binary Files
**Description**: Implementation of LSD (Least Significant Digit) Radix Sort algorithm operating on binary files for large-scale data sorting.

**Key Features**:
- **LSD Radix Sort** implementation
- Binary file processing
- Efficient sorting for large datasets
- Stable sorting algorithm
- Comprehensive unit tests

**Skills Demonstrated**:
- Advanced sorting algorithms
- Binary file I/O
- Algorithm complexity analysis
- Large-scale data processing

**Why This Impresses Recruiters**:
Radix sort is a non-comparative sorting algorithm with O(n) complexity for fixed-length keys. This shows understanding of specialized sorting algorithms and binary data processing.

---

### 4. F25P4ATC - Air Traffic Control with SkipList & Bintree
**Description**: Air traffic control simulation using SkipList for name-based access and Bintree for spatial indexing of aircraft.

**Key Features**:
- **SkipList** for efficient name-based operations
- **Bintree** for spatial indexing of aircraft
- Collision detection algorithms
- Range queries for spatial searches
- Comprehensive unit tests

**Skills Demonstrated**:
- Advanced data structures (SkipList, Bintree)
- Spatial indexing
- Collision detection algorithms
- Multi-structure integration

**Why This Impresses Recruiters**:
SkipLists provide O(log n) operations with simpler implementation than balanced trees. Bintrees are used in spatial partitioning. This shows ability to integrate multiple advanced data structures for complex problems.
