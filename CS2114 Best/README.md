## 📋 Overview

These 5 projects demonstrate mastery of fundamental and advanced data structures, including arrays, linked lists, queues, sets, and recursive algorithms. Each project includes comprehensive testing and clean, well-documented code.

## 🎯 Projects

### 1. DisplayCollection2 - Bag Data Structures
**Description**: Implementation of bag data structure using both array and linked list approaches, with a GUI game interface.

**Key Features**:
- **Dual implementation**: Array-based and linked-list based bags
- Generic type support (Java generics)
- Random element selection
- Comprehensive unit tests for both implementations
- GUI game integration (WhackAShape)

**Skills Demonstrated**:
- Multiple implementation approaches for same problem
- Generic programming
- Data structure design patterns
- GUI integration

**Why This Impresses Recruiters**:
Shows ability to implement the same abstract data type in multiple ways, demonstrating deep understanding of trade-offs between array and linked implementations.
