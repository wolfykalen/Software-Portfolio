# CS1114 Best Projects - Introduction to Computing

**Course**: CS1114 - Introduction to Computing  
**Focus**: Object-oriented programming, testing, game development  
**Author**: Kalen Dacosta (kalendaco)

## 📋 Overview

These 5 projects demonstrate strong fundamentals in Java programming, extensive unit testing, and problem-solving. Each project includes comprehensive test suites (400+ test lines), making this portfolio particularly suitable for QA/Testing roles.

## 🎯 Projects

### 1. program01 - Scavenger Hunt
**Description**: Maze navigation game where a Jeroo robot navigates through obstacles to collect flowers and disable nets.

**Key Features**:
- Object-oriented design with multiple classes
- Complex pathfinding algorithms
- **445 lines of comprehensive unit tests**
- Edge case handling for various maze configurations

**Skills Demonstrated**:
- Object-oriented programming
- Algorithm design and implementation
- Extensive testing methodology

---

### 2. program05 - Schelling Simulation
**Description**: Agent-based simulation modeling residential segregation patterns based on satisfaction thresholds.

**Key Features**:
- 2D grid-based simulation
- Dynamic agent relocation based on neighbor satisfaction
- **408 lines of unit tests**
- Configurable parameters (satisfaction threshold, red line)

**Skills Demonstrated**:
- Simulation modeling
- 2D array manipulation
- Statistical analysis of agent behavior

---

### 3. program09 - CleverBrain (Tetris AI)
**Description**: AI brain for Tetris game that calculates optimal piece placements to maximize score and minimize holes.

**Key Features**:
- Heuristic-based decision making
- Board state evaluation
- Piece rotation and position optimization
- Unit testing for move selection

**Skills Demonstrated**:
- AI/algorithm design
- Heuristic evaluation
- Game development concepts

---

### 4. program10 - StudentBoard (Tetris Board)
**Description**: Complete Tetris board implementation with piece placement, row clearing, and collision detection.

**Key Features**:
- 2D grid management
- Piece collision detection
- Row clearing mechanics
- **263 lines of comprehensive tests**
- Multiple piece types and rotations

**Skills Demonstrated**:
- 2D array manipulation
- Game logic implementation
- Edge case handling

---

### 5. program11 - WeatherBureau
**Description**: Weather data processing system that reads station data, calculates statistics, and finds extreme values.

**Key Features**:
- File I/O and data parsing
- Data aggregation and analysis
- Station management system
- Unit testing for data operations

**Skills Demonstrated**:
- File processing
- Data structures for station management
- Statistical calculations

## 🧪 Testing Emphasis

All projects include extensive unit tests using JUnit and AssertJ, demonstrating:
- Test-driven development practices

This makes these projects particularly valuable for **QA/Testing roles**.

Each project can be run in Eclipse or IntelliJ IDEA:
1. Import the project folder
2. Locate the `Application.java` or main class
3. Run as Java Application
4. Run test classes to verify functionality
